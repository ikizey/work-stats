### small CLI app to keep track of time on projects.

```
Usage: work_stats [options]

Options:
  -d, --debug                  output extra debugging.
  -i, --input <project_name>   Add data for <project_name>.
  -o, --output                 Show all work stats.
  -e, --tail <lines>           Show last <lines> lines.
  -a, --head <lines>           Show first <lines> lines.
  -f, --filter <project_name>  Filter the work stats.
  -v, --version                Show the version number.
  -h, --help                   display help for command
```
